# Recipe Manager

A full-stack recipe management app with ingredient scaling and shopping list generation. Built with Next.js, TypeScript, Tailwind CSS, and PostgreSQL.

🔗 **Live Demo:** [coming soon]

---

## Features

- Browse and search personal recipes
- Scale ingredient quantities by serving size with automatic unit simplification
- Generate a consolidated shopping list from multiple recipes
- User authentication and personal recipe storage

---

## Tech Stack

**Frontend:** Next.js 16 (App Router), React Server Components, TypeScript, Tailwind CSS  
**Backend:** Next.js Route Handlers, Prisma ORM  
**Database:** PostgreSQL (Supabase)  
**Auth:** (NextAuth / JWT — to be added)  
**Deployment:** Vercel + Supabase
